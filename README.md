# Виртуальная Лаборатория (Interactive Physics Lab)

## Описание проекта
Данный проект представляет собой веб-приложение для проведения виртуальных лабораторных работ по физике. Оно разработано для помощи студентам в изучении физических процессов путем визуализации и интерактивного взаимодействия. В приложении полностью реализован красивый интерфейс, соответствующий современным требованиям веб-дизайна.

## Реализованный функционал
- **Главная страница**: Каталог доступных симуляций с фильтрацией по разделам физики, детальным поиском и индикацией платных/бесплатных тем.
- **Интерактивные симуляции**: 
  - Наклонная плоскость (Динамика)
  - Движение по кругу (Кинематика)
  - Момент силы (Статика)
  - Баллистика (Механика)
  - Маятник (Колебания)
  - Упругий удар (Импульс)
  - Гравитация (Астрофизика)
  - Закон Гука (Динамика)
  - Идеальный газ (Молекулярная физика)
- **Кастомные темы**: Добавление собственных теоретических справок-модулей с использованием WYSIWYG редактора.
- **Панель управления (Admin Panel)**: Управление пользователями и контентом сайта.
- **Обработка ошибок и пустые состояния**: Реализована устойчивая работа приложения при некорректных данных (например, когда поиск не дал результатов или доступ заблокирован).

## Сборка и запуск
1. Установите зависимости: `npm install`
2. Режим разработки: `npm run dev`
3. Сборка для продакшена: `npm run build`
4. Запуск линтера: `npm run lint`
