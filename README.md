# Разработка электронного пособия по физике с интерактивным взаимодействием с пользователем

## Описание проекта
Данный проект представляет собой интерактивное веб-приложение для проведения виртуальных лабораторных работ по физике. Приложение разработано (в рамках дипломного проекта) с целью предоставления студентам визуальных и интерактивных инструментов для освоения сложных физических процессов.

## Цели и задачи
- **Цель:** Создать доступную виртуальную среду для изучения физики за счет проведения интерактивных симуляций.
- **Задачи:**
  - Реализация интерфейса с поддержкой каталога тем и личного кабинета.
  - Разработка физических симуляций (Динамика, Кинематика, Механика и т.д.) с расчетами в реальном времени.
  - Внедрение панели администратора для управления контентом лекций и симуляций.
  - Обеспечение надежности работы при любых крайних случаях и пустых поисковых запросах.

## Используемый стек технологий
- **Frontend Framework:** React 18
- **Инструмент сборки:** Vite
- **Роутинг:** React Router DOM v6
- **Управление состоянием и темами:** хуки React (useState, useEffect, Context API)
- **Графика и физика:** HTML5 Canvas API
- **Анимации и UI:** Framer Motion, Vanilla CSS (Глассморфизм), Lucide-React
- **Локальное хранилище данных:** localStorage (с возможностью миграции на PHP/MySQL)

## Архитектура приложения
Архитектура приложения построена по классическому клиентскому паттерну Single Page Application (SPA). Проект строго разделен на три слоя:
1. **Слой данных:** Изолированные сервисы (например, `/src/api`) для работы с localStorage/базой данных.
2. **Слой бизнес-логики:** Кастомные хуки (например, `useTopics.js`) и Context API (`AuthContext`, `ThemeContext`).
3. **Слой представления (UI):** Разделение на страницы (`/pages`) и переиспользуемые UI-компоненты (`/components`), включая изолированный модуль симуляций (`/components/simulations`).

## Инструкция по запуску
1. Установите зависимости: `npm install`
2. Запустите проект в режиме разработки: `npm run dev`
3. Соберите проект для продакшена: `npm run build`
4. Предпросмотр продакшен-сборки: `npm run preview`
5. Проверка качества кода: `npm run lint`
